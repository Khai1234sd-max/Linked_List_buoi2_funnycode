Giải thích tại sao khi a dùng 
for (int i = 0; i <= index; i++) {
            temp = temp.next;
        } 
thì temp nó là node tại vị trí index
còn nếu anh dùng ntn:
for (int i = 0; i < index; i++) {
            temp = temp.next;
        } 
thì temp nó là node trước vị trí index


-------
